# Agro Forte

![Banner Agro Forte](https://images.unsplash.com/photo-1500937386664-56d1dfef3854?q=80&w=1600&auto=format&fit=crop)

## Sobre o Projeto

**Agro Forte** é um site institucional que promove a **produção agrícola sustentável**, integrando tecnologia, inovação e preservação ambiental. O projeto apresenta conceitos modernos de agronegócio que respeitam o meio ambiente e incentivam práticas sustentáveis.

O site foi construído usando **HTML e CSS puro**, sendo responsivo e adaptável a diferentes tamanhos de tela.

---

## Funcionalidades

- **Seção Hero**: Banner principal com título, descrição e botão de navegação.
- **Produção Sustentável**: Cards com imagens e informações sobre:
  - **Agricultura Moderna**  
    ![Agricultura Moderna](https://images.unsplash.com/photo-1464226184884-fa280b87c399?q=80&w=1200&auto=format&fit=crop)  
    Uso de tecnologia para aumentar a produtividade no campo com eficiência e sustentabilidade.
  
  - **Preservação Ambiental**  
    ![Preservação Ambiental](https://images.unsplash.com/photo-1501004318641-b39e6451bec6?q=80&w=1200&auto=format&fit=crop)  
    Proteção das florestas, rios e biodiversidade para garantir equilíbrio ecológico.

  - **Energia Verde**  
    ![Energia Verde](https://images.unsplash.com/photo-1506744038136-46273834b3fb?q=80&w=1200&auto=format&fit=crop)  
    Investimento em energia limpa e renovável para um futuro mais sustentável.

- **Seção Sobre**: Explica a relação entre agronegócio moderno e preservação ambiental.
- **Responsividade**: Layout ajustável a diferentes dispositivos, incluindo celulares e tablets.
- **Design Limpo**: Cores suaves, tipografia clara e efeitos de hover nos cards.

---

## Tecnologias Utilizadas

- **HTML5**: Estrutura do site.
- **CSS3**: Estilização e responsividade.
- **Imagens**: Fotos gratuitas do [Unsplash](https://unsplash.com/).
- **Fonte**: Arial, sans-serif.
