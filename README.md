## Sobre o Projeto

**Agro Forte** é um site institucional que apresenta o conceito de **produção agrícola sustentável**, unindo tecnologia, inovação e preservação ambiental. O objetivo é destacar práticas modernas de agricultura que respeitam o meio ambiente e promovem a sustentabilidade no agronegócio.

O site foi desenvolvido com **HTML e CSS puro**, sendo responsivo e otimizado para diferentes dispositivos.

---

## Funcionalidades

- **Seção Hero**: Banner principal com mensagem de impacto e botão de navegação para conhecer mais sobre produção sustentável.
- **Produção Sustentável**: Cards informativos sobre:
  - Agricultura Moderna
  - Preservação Ambiental
  - Energia Verde
- **Sobre**: Informações sobre a integração entre agronegócio e meio ambiente.
- **Responsividade**: Layout adaptável a telas de diferentes tamanhos, incluindo dispositivos móveis.
- **Design Atraente**: Uso de cores suaves, imagens impactantes e efeitos visuais simples (hover nos cards).

---

## Tecnologias Utilizadas

- **HTML5**: Estrutura do site
- **CSS3**: Estilização e responsividade
- **Fontes**: Arial, sans-serif
- **Imagens**: Fotos gratuitas da plataforma [Unsplash](https://unsplash.com/)
